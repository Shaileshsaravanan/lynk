Error: Flask Server Not Running

The application could not connect to the Flask backend server. This is required for the application to function properly.

What happened?

The application failed to start its internal server. This could be due to:

  • Missing Python dependencies
  • Port 5001/5002 already being used by another application
  • Insufficient permissions
  • Corrupted application files

What can you do?

  1. Try restarting the application
  2. If the problem persists, reinstall the application
  3. Contact support if the issue continues
