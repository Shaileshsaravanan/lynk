Error: Flask Server Not Running

The application could not connect to the Flask backend server. This is required for the application to function properly.

Troubleshooting Steps:

  1. Make sure the Flask server is running. You can start it with: python app.py
  2. Check if there are any error messages in the terminal where you started the application
  3. Verify that port 5001 is not being used by another application on 127.0.0.1
  4. Try restarting the application with: npm run dev
